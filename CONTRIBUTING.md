# Contributing

All changes to this repository are made via
[pull requests](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/about-pull-requests).

For detailed setup and workflow instructions, see:

- [Editing Conventions](docs/meta/editing_conventions.md) — markdown links, Excalidraw format,
  file naming
- [Local Dev Environment](docs/meta/local_dev_environment.md) — uv, `mkdocs serve`, pre-commit
  hooks
- [Editing on GitHub](docs/meta/editing_on_github.md) — contributing directly from the browser
- [Using Obsidian](docs/meta/using_obsidian.md) — opening the repository as an Obsidian vault
