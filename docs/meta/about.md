This site is the personal knowledge base (PKB) of
[Felix Schindler](https://github.com/ftschindler), containing notes on running Linux in
various circumstances. It is a git-backed static site built from Markdown files and
[Excalidraw](https://excalidraw.com) diagrams.
See [Tech Stack](tech_stack.md) for more information, or
[PKB Tech Stack](../done/20260520-pkb-tech-stack.md) for the full design rationale.

## How to contribute

The repository supports three equally valid ways of working — pick whichever suits you:

1. **[Edit on GitHub](editing_on_github.md)** — make changes directly in the browser, no local
   setup required.
2. **[Local dev environment](local_dev_environment.md)** — clone the repository, preview with
   `mkdocs serve`, use any text editor.
3. **[Obsidian](using_obsidian.md)** — open the repository as an Obsidian vault with full editing
   UX and inline Excalidraw diagrams.

All workflows follow the same [editing conventions](editing_conventions.md).
